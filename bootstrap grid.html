We can be more specific in having number of column with same column space according to differnt screen size such as Deaktop - lg, Tablet - md and Mobile - sm.The Bootstrap Screen Size difinition - Extra small "col-xs-x" < 576px Small "col-sm-x" ≥ 576px Medium "col-sm-x" ≥ 768px Large "col-lg-x" ≥ 992px Extra large "col-xl-x" ≥ 1200px

Example below will demonstarte the Bootstrap response to the "View Port" or Screeen Size according to our definition "col-lg-3" - Means Display 4 columns whe the Screen Size is Large because 12/3 = 4. "col-md-4" - Means Display 3 columns whe the Screen Size is Medium because 12/4 = 3. "col-sm-6" - Means Display 2 columns whe the Screen Size is Large because 12/6 = 2. When the Screen Size is smaller than 576px then only one column will be display
